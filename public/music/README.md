# Background Music Files

This directory is where you can add your own instrumental background music files for the reading experience.

## How to Add Your Music Files

1. **Add your music files** to this directory (`public/music/`)
2. **Name them** as: `track1.mp3`, `track2.mp3`, `track3.mp3`, etc.
3. **Update the track names** in the PDFReader component to match your files

## File Requirements

- **Format**: MP3 (recommended) or other web-compatible audio formats
- **Duration**: Any length (will loop automatically)
- **Volume**: Mixed at moderate levels (app will set to 30% volume)
- **Content**: Instrumental music works best for reading
- **Quality**: 128kbps or higher recommended

## Supported Audio Formats

- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)
- M4A (.m4a)

## Usage

Once you add your music files, users can select and play them while reading PDFs using the music controls in the PDF reader interface.

## Example

If you add files like:
- `my-favorite-song.mp3`
- `relaxing-music.mp3`
- `study-music.mp3`

The app will automatically detect and allow users to select from these tracks.
